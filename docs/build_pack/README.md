# CareLoop HK — AI Agent Build Pack

Selected idea: **CareLoop HK**, a remote chronic-care monitoring platform for elderly Hong Kong patients between clinic visits.

This is not an AI doctor, diagnosis tool, treatment recommender, or generic elderly chatbot.

Core demo:
```txt
elderly patient dashboard
→ daily Cantonese check-in
→ wearable/vital data
→ deterministic risk rules
→ nurse review queue
→ caregiver alert
→ weekly clinician PDF
→ FHIR-style export
→ HONESTY.md
```

Use files in order:
1. `01_careloop_hk_full_build_spec.md`
2. `02_polish_demo_and_winning_criteria.md`
3. `03_technical_architecture_and_stack.md`
4. `04_demo_script_and_pitch.md`
5. `05_HONESTY_template.md`

Core rule: AI is only for summarisation and caregiver-friendly wording. Risk classification must be deterministic, explainable, and rule-based.
