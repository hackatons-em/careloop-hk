# Miruwa — one-pager (sales leave-behind)

*Remote chronic-care monitoring, between clinic visits. Bilingual EN / 繁體中文.*

## English

**The gap.** Chronic-care patients are seen only now and then. Between visits,
deterioration goes unseen — until it's an emergency readmission.

**Miruwa.** A daily check-in over **WhatsApp** — patients can simply send a
**Cantonese voice note**. Miruwa transcribes and understands it, and a
**deterministic, explainable rule engine** (not AI diagnosis) surfaces **only the
patients who need a nurse**, with the exact rule and evidence.

**For the nurse / care team:**
- One triage queue — only the exceptions, not 200 charts.
- Trend tracking: weight, blood pressure, activity, medication adherence.
- Consent-gated caregiver / family alerts.
- Every flag and action on an **append-only audit trail**.

**For the program lead:**
- Measurable outcomes — check-in response rate, alert volume, time-to-acknowledge —
  and a one-click board PDF.

**Safe by design.** Monitoring support, **not a medical device, not diagnosis**.
Severity is decided by a fixed, auditable rule set — same data in, same result out.
One isolated instance per organization; PDPO-aligned; data export on request.

**Built for Hong Kong.** Bilingual, WhatsApp-native, live today. Fits DHC /
Chronic Disease Co-Care nurse follow-up as the between-visit layer.

**Pilot offer.** A [12-week] pilot, up to [50] patients, success measured on a
**≥70% check-in response rate**. → **Request a pilot.**

## 繁體中文

**問題。** 慢性病患者只在覆診時被看顧。覆診之間的惡化往往無人察覺 —— 直到變成緊急再入院。

**Miruwa。** 病人每日透過 **WhatsApp** 報到，甚至只需傳一段**廣東話語音**。Miruwa
即時轉錄並理解內容，再由**可解釋的確定性規則引擎**（並非 AI 診斷）**只篩選出需要護士跟進的病人**，
並附上觸發的規則與證據。

**給護理團隊：**
- 一條分流工作列 —— 只顯示需要關注的少數，而非全部病歷。
- 趨勢追蹤：體重、血壓、活動量、服藥依從性。
- 經同意的照顧者／家人警示。
- 每個警示與操作均有**唯讀審計紀錄**。

**給計劃負責人：**
- 可量度的成效 —— 報到回覆率、警示數量、確認時間 —— 一鍵生成董事會報告 PDF。

**安全為本。** 屬監測支援，**並非醫療器械、並非診斷**。嚴重程度由固定、可審計的規則決定 ——
相同數據、相同結果。每間機構獨立部署；符合《個人資料（私隱）條例》；可按要求匯出資料。

**為香港而建。** 雙語、WhatsApp 原生、現已運作。可作為地區康健中心／慢性疾病共同治理計劃
護士跟進的「覆診之間」一層。

**試驗計劃。** 為期 [12 週]、最多 [50] 名病人，成效以**報到回覆率 ≥ 70%** 衡量。
→ **預約試驗計劃。**
